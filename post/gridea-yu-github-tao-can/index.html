Gridea与Github套餐

2019-10-07 / 1 min read

把原来hexo删了，依然是基于GitHub，重新使用Gridea翻新了这个博客。教程参考了Gridea官网和B站上的一个视频

Gridea这个软件用起来还是蛮舒服的，这套系统我觉得有如下优缺点：
优点：

  1. Gridea软件用起来舒服，适合不会编程的小白
  2. 后期维护简单，GitHub全包了，稳定
  3. 成本低廉，只需要购买域名（甚至可以用GitHub的二级域名）

缺点：

  1. 主题还是太少，毕竟是个新的项目，活跃度一般
  2. GitHub国内连接速度一般，不能放大图片
  3. 有些许被墙的风险（微软收购后应该还好）
  4. 基于PC和MAC，无移动端，也没有网页端；虽说多数情况下都是用PC码字，也能通过网盘同步，但在一些特殊情况下有些不方便
  5. 静态页面，渲染要一段时间
